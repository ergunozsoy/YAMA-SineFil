# SİNEFİL

Sinefil, sevilen filmleri, yönetmenleri ve unutulmaz sahneleri bir araya getiren kişisel bir sinema arşivi prototipidir.

## Konsept

“Sinefil” sözcüğü burada hem sinema tutkunu anlamında hem de “sinema + fil” çağrışımıyla ele alınır. Ana karakter, boş bir sinema salonunda tek başına film izleyen bir fildir.

## İçerik

- Ana sayfa ve özgün fil maskotu
- Sessiz sinema koleksiyonu
- Film noir koleksiyonu
- Yönetmen sayfaları
- Film sayfaları
- Unutulmaz sahneler bölümü
- Mobil uyumlu tasarım
- GitHub Pages ile doğrudan yayınlanabilir yapı

## Çalıştırma

`index.html` dosyasını tarayıcıda açın.

Daha sağlıklı yerel önizleme için:

```bash
python -m http.server 8000
```

Ardından tarayıcıda `http://localhost:8000` adresine gidin.

## GitHub Pages

1. Dosyaları GitHub reposuna yükleyin.
2. Repository → Settings → Pages bölümünü açın.
3. Source olarak `Deploy from a branch` seçin.
4. `main` ve `/root` seçeneklerini seçip kaydedin.

## Fotoğraflar

`assets/images/` klasöründeki sinema salonu fotoğrafları kullanıcı tarafından sağlanmıştır.
