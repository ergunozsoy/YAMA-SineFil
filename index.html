Sinefil’in koltuğu

Sinefil's seat

Sinefils Sessel

“Burada filmler puanlanmaz; hatırlanır, yeniden görülür ve konuşulur.”
“Here films are not rated; they are remembered, revisited and talked about.”
„Hier werden Filme nicht bewertet; sie werden erinnert, wiedergesehen und besprochen.“

Sinefil, bir veri tabanından çok kişisel bir sinema defteridir. Sevilen yönetmenleri, filmleri, sahneleri, renkleri, sesleri ve seyir anılarını bir araya getirir.

Sinefil is less a database than a personal cinema notebook. It brings together beloved directors, films, scenes, colours, sounds and the memories of watching.

Sinefil ist weniger eine Datenbank als ein persönliches Kino-Notizbuch. Es versammelt geliebte Regisseure, Filme, Szenen, Farben, Klänge und die Erinnerungen des Zuschauens.

Sinefil ne demek?

What does “Sinefil” mean?

Was bedeutet „Sinefil“?

“Sinefil” sinemaya tutkuyla bağlı kişiyi anlatır. Bu projede sözcük, Türkçedeki “sinema” ve “fil” çağrışımıyla yeniden yorumlanıyor: karanlık bir salonda, en sevdiği filmi tek başına izleyen bir fil.

“Sinefil” describes someone passionately devoted to cinema. In this project the word is reinterpreted through its echo of the Turkish words “sinema” and “fil” (elephant): an elephant watching its favourite film alone in a dark auditorium.

„Sinefil“ bezeichnet einen leidenschaftlich dem Kino verbundenen Menschen. In diesem Projekt wird das Wort durch den Anklang an die türkischen Wörter „sinema“ und „fil“ (Elefant) neu gedeutet: ein Elefant, der in einem dunklen Saal allein seinen Lieblingsfilm ansieht.

Repo; GitHub Pages üzerinde doğrudan yayınlanabilecek biçimde, bağımlılıksız HTML, CSS ve küçük bir JavaScript dosyasıyla hazırlanmıştır.

The repository is built with dependency-free HTML, CSS and a small JavaScript file, ready to be published directly on GitHub Pages.

Das Repository ist mit abhängigkeitsfreiem HTML, CSS und einer kleinen JavaScript-Datei aufgebaut und kann direkt auf GitHub Pages veröffentlicht werden.
